Usability Problems

      Learnability: easy to perform the basic task of searching - difficult to do anything else
      Efficiency: very low because the user has to search for exactly what they are looking for -
      no tabs or headers to give users a quick way to access information
      Memorability: vacuously high because the site has few features - not much to forget how to do
      Accessibility: WAVE only found a few errors and alerts, likely due to the barren nature of the site.
      The main accessibility issue was the presence of > 0 tabindices, which affects the keyboard accessibility.
      I agree with the findings because inspecting the HTML shows that many elements have a positive tabindex.
